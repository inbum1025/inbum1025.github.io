# Inbum Heo

Personal GitHub Pages site for Inbum Heo.

## Local Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

Target URL:

```text
https://inbum1025.github.io/
```

Use a repository named exactly:

```text
inbum1025.github.io
```

Enable GitHub Pages from the repository settings and set the source to the root of the default branch.
